# Car Parts Scraper

Playwright-based scraper for Inter Cars parts aggregation.

## Setup

```bash
cd car-parts-scraper
npm install
npx playwright install chromium
```

## Run

```bash
INTERCARS_EMAIL=your@email.com INTERCARS_PASSWORD=yourpass npm run scrape
```

Or from the monorepo root:

```bash
INTERCARS_EMAIL=your@email.com INTERCARS_PASSWORD=yourpass npm run scrape
```
