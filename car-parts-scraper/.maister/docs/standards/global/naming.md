## Naming Conventions (Project-Specific)

### camelCase File Names

TypeScript source files use camelCase naming.

```
interPartsParser.ts
apcatScraper.ts
autoPartnerParser.ts
```

### kebab-case Directory Names

Feature directories use kebab-case naming.

```
src/inter-parts/
src/auto-partner/
src/apcat/
```
